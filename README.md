# Tenzin Tsundue Portfolio Website

A slick static personal portfolio site built with HTML, CSS, and JavaScript.

## Files

- `index.html`
- `style.css`
- `script.js`

## How to use

1. Copy these files into your GitHub Pages repository: `tenzin3.github.io`
2. Commit and push to GitHub.
3. Enable GitHub Pages from repository settings if needed.
4. Visit `https://tenzin3.github.io/`

## Edit these links

In `index.html`, replace placeholder LinkedIn/other links when you have the exact profile URLs.
